Assignment 2 Definitions and Terms

How to define a table?
You can define a table by using the <table>tag. This will define a table in your code, but in order to make the proper tables we usually see, we need to use some other tags which will be described below.

<tr>
This tag defines a row in the table. Use it when you want to add a single row to the table.

<th>
This tag allows us to add a header to the table. You should use this when making labels describing what the data in the table is.

<td>
This is a regular cell in the table. Use this when you want to input the data for your table.

<thead>
This tag is used to group the head of the elements in the table. It makes it easier to style contents of these elements. Kind of lets the browser know where is the top of the table.

<tfoot>
This tag is used to group the bottom elements of the table. It makes it simpler to group and style if you have a total or some summation for your table. It also helps the browser find the bottom or end of the table.

<caption>
This allows you to caption your table. Kind of like a table's title per se.

What is the difference between colspan and rowspan?
colspan defines the nubmer of columns a cell should take up, while rowspan defines the number of rows a cell should take up.

What is the difference between cell spacing and padding?
Cell spacing defines how much pixels should be in between different cells, while padding tells us how much space should be between the cell's walls and the contents within that cell.

How do you define table borders? Control border widths?
You can define table borders by adding border to the opening tag of the table like this: <table border="1">. This tells us the browser we want to show borders. You can control border widths by using css and editing the attributes border-width,border-left-width,border-right-width,... and more.

Definitions

<DOCTYPE>
A declaration that instructs the web browser about what version of HTML the page is written in.

How to use <ol>, <ul>, <dl>
<ol>
A tag for an ordered list. To use this you will need to use the <li> tag for each of your items in the list. They'll then be numbered according to the order you put them in your html.
<ul>
A tag for an unordered list. Just like <ol>, you need to use the <li> tag for each of your items in the list. They'll be then bulleted in the list with no numbering.
<dt>
A tag for a description list. This list is a little different than the others. You will need to use <dt> for the term's name. Then you'll need to use <dd> for the description of the term.

Difference between text editor and word processor?
Text editors are for modifying plain text like ASCII or UTF-8. It's mostly used for programming languages like HTML or Javascript, etc. A word processor is just used for typing up documents or just editting text that you would print.

What is the client-server model?
The client-server model is a type of I guess relationship in applications. Basically the server allows the client to create certain functions, and these functions are done made on the request of the client. (I think of it kind of like supply and demand)

FTP
File Transfer Protocol. This is the standard network protocol that that computers use to transfer files between servers and clients on a network
HTTP
Hyper Text Transfer Protocol. This is probably the most common protocol used on the web. It determines how things are transmitted or formatted and also tells the servers or browsers what it should do in response.
TCP/IP
Transmission Control Protocol or Internet Protocol. It's a protocol for sending packets of data across two connected programs.
